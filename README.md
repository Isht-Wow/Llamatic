# Llamatic

**Lightweight local llama.cpp model manager**

---

## Overview

Llamatic is a fast, minimal Node.js server that manages your local `llama.cpp` models.  
It allows you to run large language models locally without relying on Electron or bloated frameworks.  

Built with Google Antigravity.

---

## Features

- Local model management and inference
- Runs fully on macOS, Linux, or Windows (Node.js runtime embedded)
- Supports background execution — no terminal required
- Built-in browser interface with proper favicon
- Minimal dependencies — everything you need is packaged
